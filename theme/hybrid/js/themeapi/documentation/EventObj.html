eventWatcher

boolean eventWatcher(<string> prop, <string> oldval, <string> newval)
    Watches eventprop for change. When eventprop registers a change in value an event array is selected based on the newval property. This event array is then looped through and executed. The eventprop is then reset.
    Parameters:
      prop - Event type
      oldval - Event identifier
      newval - Event identifier
    Returns:
      Always returns false

ieCheck
